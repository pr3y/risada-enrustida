# risada-enrustida
base5 encoding

## Usage
```
Usage:
  target/release/risada-enrustida [OPTIONS]

Decodificador de RISADAs.

Arguments:
  -h,--help             Show this help message and exit
  -d,--decode DECODE    Usa opcao de decodar
  -e,--encode ENCODE    Usa opcao de encodar
```

## Compiling
`cargo build --release`
